## Run Locally

Clone the project

```bash
git clone https://github.com/Sa-YoorHeadley/Password-Strength-Meter.git
```

Go to the project directory

```bash
cd Password-Strength-Meter
```

Run using Live Server
```
Alt + L 
Alt + O
```
